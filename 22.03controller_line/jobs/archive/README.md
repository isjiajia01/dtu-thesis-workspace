# Archived Jobs

This folder stores superseded HPC entrypoints that are no longer part of the retained paper-facing experiment set.

Archive categories:

- `runtime/`
  - historical runtime-policy, cap, reopt, and fixed-vs-dynamic time-budget sweeps
- `ood/`
  - historical OOD sensitivity sweeps and non-retained depot-side runtime variants
- `tuning/`
  - abandoned controller branches, `mt3` / fleet probes, duplicate tuning paths, and older rerun helpers

Archived jobs are kept only for traceability.
If one of them needs to become active again, the thesis scope documents must be updated in the same task.
