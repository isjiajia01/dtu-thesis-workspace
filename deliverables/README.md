# Deliverables

This folder collects exported artifacts that are useful to view or share but are not part of the active source tree.

## Current layout

- `html/` — standalone HTML exports such as presentations, dashboards, and demos.

## Current HTML artifacts

- `html/22.03-multi-depot-dashboard.html` — large multi-depot dashboard export associated with the `22.03controller_line` line.
- `html/herlev-1044-orders-dashboard.html` — smaller single-scenario dashboard export for the 1044-order view.
- `html/mover-multi-depot-dashboard-scroll.html` — alternate multi-depot dashboard export with a scroll-oriented layout.
- `html/presentation.html` — thesis progress / professor-meeting presentation deck.

## Placement rules

Keep source code, analysis scripts, and manuscript files in the thesis phase folders.
Put rendered shareable outputs here when they would otherwise clutter the repository root or active thesis directories.

## Cleanup rules

- Prefer descriptive names; avoid Finder-style `* 2` filenames.
- Remove exact duplicates rather than keeping parallel copies in multiple folders.
- If an artifact becomes inactive but should still be preserved, move it to `archive/` instead of leaving it mixed with active deliverables.
